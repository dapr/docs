## Description

Check if mTLS is enabled in a Kubernetes cluster

## Usage

```bash
dapr mtls [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--help`, `-h` | | | Help for mtls |
| `--kubernetes`, `-k` | | `false` | Check if mTLS is enabled in a Kubernetes cluster |
