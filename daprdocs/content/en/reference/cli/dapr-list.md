## Description

List all Dapr instances

## Usage

```bash
dapr list [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--help`, `-h` | | | Help for list |
| `--kubernetes`, `-k` | | `false` | List all Dapr pods in a k8s cluster |
