## Description

Stop Dapr instances and their associated apps in self-hosted mode

## Usage

```bash
dapr stop [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--app-id` | | | The application id to be stopped |
| `--help`, `-h` | | | Print this help message |
