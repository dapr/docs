## Description

Get the version and git commit data for `dapr-cli` and `daprd` executables.

## Supported platforms

- [Self-Hosted]({{< ref self-hosted >}})

## Usage
```bash
dapr build-info
```

## Related facts

You can get `daprd` build information directly by invoking `daprd --build-info` command.
