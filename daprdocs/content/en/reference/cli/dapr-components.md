## Description

List all Dapr components

## Usage

```bash
dapr components [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--help`, `-h` | | | Help for components |
| `--kubernetes`, `-k` | | `false` | List all Dapr components in a k8s cluster |
